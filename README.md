# Readings Management

Readings Management is a project from Udacity's React Nanodegree course. This is a bookshelf app that lets you to organize books that you have read, that is current reading or want to read.

## How to get started

To setup this app, you will need to follow the instructions below:

1 - Install all project dependencies with `npm install`

2 - Start the development server with `npm start`

It's done! You should have Readings Management running straight way...
