# Readings Management

Readings Management is a project from Udacity's React Nanodegree course. This is a bookshelf app that lets you to organize books that you have read, that is current reading or want to read.

## How to get started

To get started developing right away:

* install all project dependencies with `npm install`
* start the development server with `npm start`
